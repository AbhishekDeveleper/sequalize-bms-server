# Node.js CRUD with Sequelize

This is a basic Node.js server application that demonstrates CRUD operations using Sequelize ORM with a MySQL database.

## Features

- Create, Read, Update, and Delete (CRUD) operations on a Book model.
- Uses Sequelize for ORM with MySQL.
- Basic Express.js server setup.

## Prerequisites

- Node.js
- MySQL Database
- Sequelize
- MySQL2
